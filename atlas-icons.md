## Using icons

### React

- Always import icons from `@diligentcorp/atlas-react-icons/dist/esm/lens/[IconName].js`
- Example: `import EditFileIcon from '@diligentcorp/atlas-react-icons/dist/esm/lens/EditFile.js'`
- Never use barrel files for icons, always import icons individually

### Angular

```typescript
import { Component } from "@angular/core";
import { EditFileIconComponent, AddCircleIconComponent } from "@diligentcorp/atlas-angular-icons";

@Component({
  // ...component metadata
  imports: [EditFileIconComponent, AddCircleIconComponent],
  template: `
    <atlas-edit-file-icon></atlas-edit-file-icon>
    <atlas-add-circle-icon></atlas-add-circle-icon>
    <!-- When nesting into a button, wrap it with `mat-icon` -->
    <button mat-flat-button color="primary">
      <mat-icon>
        <atlas-add-circle-icon></atlas-add-circle-icon>
      </mat-icon>
    </button>
  `,
})
export class MyComponent {
}
```
