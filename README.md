# Community V2 Version B

This prototype is deployed via [VibeSharing](https://vibesharing.app).

## Getting Started

1. Clone this repo:
   ```bash
   git clone https://github.com/vibesharing-prototypes/community-v2-version-b
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run locally:
   ```bash
   npm run dev
   ```

4. Make your changes in `src/`

5. Push to deploy:
   ```bash
   git add .
   git commit -m "Update prototype"
   git push
   ```

Your changes will be live in ~30 seconds.

## Deployment

Netlify builds with `npm run build` and serves `dist` per `netlify.toml`.
